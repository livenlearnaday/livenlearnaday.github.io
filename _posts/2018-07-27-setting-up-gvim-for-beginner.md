In this article you will find very basic gvim setup (these commands can be found in gvim help files and various sources online) that will show you how to set gvimrc file to:

- set colorscheme
- set tab setting
- set autocomplete for various programming languages
- start gvim window at a certain position in the window
- start gvim gui with a defined size

---
**NOTE**

Before setting the gvimrc file, you have to have the file in your "C:\Users\username". In my case, it's "C:\Users\pc".
